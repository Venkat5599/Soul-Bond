# NFT Images Folder

## How to Add NFT Images:

1. Add your NFT images to this folder
2. Name them: `nft-1.png`, `nft-2.png`, `nft-3.png`, etc.
3. Recommended size: 800x800 pixels
4. Format: PNG or JPG

## Current Images:

The app will randomly pick one of these images when a proposal is accepted:
- nft-1.png
- nft-2.png
- nft-3.png
- nft-4.png
- nft-5.png

## To Add More:

1. Save your image as `nft-6.png` (or next number)
2. Update the array in `ProposalInbox.jsx` if you want to add more than 5

## Image Requirements:

- Square aspect ratio (800x800 recommended)
- Romantic/couple themed
- High quality
- File size: < 2MB each

## Example Sources:

- Create your own in Canva
- Use AI generators (Midjourney, DALL-E)
- Commission an artist
- Use stock images (with proper license)

---

**Note:** For the demo, you can use any couple/heart/love themed images you find online.
